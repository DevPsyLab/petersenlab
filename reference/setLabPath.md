# Set Lab Path.

Sets the path directory to the lab drive.

## Usage

``` r
setLabPath()
```

## Value

The object `petersenLab` with containing the path directory to the lab
drive.

## Details

Sets the path directory to the lab drive, and saves it in the object
`petersenLab`.

## Examples

``` r
# \donttest{
petersenLabPath <- setLabPath()
# }
```
