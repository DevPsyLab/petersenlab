# Compare Mixed Effect Imputation Models.

Function that compares two nested `lme()` models from multiple
imputation using likelihood ratio test.

## Usage

``` r
imputationModelCompare(model1, model2)
```

## Arguments

- model1:

  name of first `lme()` model object with multiply imputed data.

- model2:

  name of second `lme()` model object with multiply imputed data.

## Value

Likelihood ratio test for model comparison of two mixed effect
imputation models.
