# Specify Decimals.

Specify the number of decimals to print.

## Usage

``` r
specify_decimal(x, k)
```

## Arguments

- x:

  Numeric vector.

- k:

  Number of decimals to print.

## Value

Character vector of numbers with the specified number of decimal places.
