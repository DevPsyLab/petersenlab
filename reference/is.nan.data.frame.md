# NaN (Not a Number).

Check whether a value is "Not A Number" (`NaN`) in a dataframe.

## Usage

``` r
# S3 method for class 'data.frame'
is.nan(x)
```

## Arguments

- x:

  Dataframe.

## Value

`TRUE` or `FALSE`, indicating whether values in a dataframe are Not a
Number (`NA`).
